Equations of Functions
======================
These functions generate the evaluations of the eight key functions: linear, quadratic, cubic, hyperbolic, exponential, logarithmic, logistic, and sinusoidal. Using them, you can generate any of the above equation types, and then plug in any variable value to learn its evaluation.
